# Project Title
DEKADAL-Summary.

## Table of Contents
* Ten day dekadal summary. 
* get the weather for a location; 
* produce statistics over a number of days; produce total values for readings, produce averages, minimum, maximum for select list of parameters;
*  we will have to digitize readings for a station over ten days 

## Description
This project generates a dekadal (10-day) summary of weather data for a given location, including totals, averages, minimum, and maximum values for parameters like rainfall, temperature, humidity, and sunshine hours.

## Prerequisites
* Accessible wealth data 
*   Node.js
*   Javascript
